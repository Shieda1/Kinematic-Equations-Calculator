What is the kinematic equations?
The kinematic equations are a set of equations that describe the motion of an object with constant acceleration.
How to calculate kinematic equations?
To calculate kinematic equations you should use the following formula:
v = u + at
v^2 = u^2 + 2 *as
s = ut + 1/2 * at^2
Why use kinematic equations?
The kinematic equations are used in order to solve for an unknown in a problem when an object has either a constant velocity or constant acceleration.
